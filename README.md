## Skill Anatomy

Each skill follows the Agent Skills standard:
- **YAML frontmatter** — name, description, version
- **Core instructions** — when to use, workflow steps, decision trees
- **Code examples** — copy-paste ready patterns
- **References** — linked deep-dive docs, API references, tutorials
- **Scripts/templates** — executable utilities where applicable
