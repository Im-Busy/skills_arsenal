## When to Use

Use this skill when:
- Building a landing page or web app and want a proven design system
- The user mentions wanting a specific style (e.g. "make it look like Notion" or "give it a Vercel vibe")
- Generating consistent UI from AI coding agents
- Needing guardrails on color, typography, components, spacing

## How to Use

1. Pick a design system from the list above, or let the user pick a URL
2. The AI should apply that design system's principles: colors, typography, components, spacing
3. Follow the Do's and Don'tts strictly
4. Use the Agent Prompt Guide section for quick reference prompts
